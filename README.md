# DB to JSON Tool (Laravel)

## Setup
1. Make sure Valet (or another PHP server) is installed: [Laravel Valet Installation](https://laravel.com/docs/5.8/valet#installation)
2. `git clone https://github.com/nullthoughts/db-to-json-tool.git`
3. `cd db-to-json-tool` & `composer install`
4. `php artisan key:generate`
5. Edit the `.env` file to specify your DB connection details
6. If using valet: `valet link`

Then browse over to [http://dbtojsontool.test](http://dbtojsontool.test) in a browser
